# tetri
A tetris game with the focus of clearing 40 lines as fast as possible

## Follows modern tetris guidelines, including:
* 10x20 board
* [fully functional SRS system](https://harddrop.com/wiki/SRS)
* [5 piece preview](https://tetris.wiki/Piece_preview)
* [hold](https://harddrop.com/wiki/Hold_piece)
* [ghost piece](https://harddrop.com/wiki/Ghost_piece) 
* [7 bag system for queue](https://harddrop.com/wiki/Random_Generator)

[demo run](https://youtu.be/8cQf9-rwNRQ)

also has:
* basic stats like keys per second, pieces per second, and [finesse faults](https://four.lol/mid-game/finesse) 
* undo/redo system, [demo here](https://youtu.be/bYmyOCGBjFA)
* invisible board: [snippet here](https://youtu.be/qKqirIZs3-s)


Since i coded the whole thing, it is alot easier for me to mod the game. If i make the game slightly easier, i can do a lot of cool stuff like
* [beating 2 games at once](https://youtu.be/Ks0Jb1eQ0Og)
* [abusing the rotaion system to clear 40 lines without moving left or right](https://youtu.be/lHNsVhHWY1Q)
